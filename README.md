# Instacart Customer Purchase Behavior and Sales Analysis

## Project Overview
This project analyzes customer purchasing behavior and sales trends using the Instacart dataset.

The project includes:
- Data Cleaning
- SQL Analysis
- Python/Colab Analysis
- Power BI Dashboard
- Data Visualization

## Tools & Technologies
- Python
- Pandas
- SQL
- Power BI
- Google Colab

## Project Structure

├── Cleaned_Data
├── Colab_Notebook
├── Power BI Reports
├── Screenshots
├── SQL_Output
└── Datasets

## Key Insights
- Customer purchasing patterns
- Frequently bought products
- Sales trends
- Order frequency analysis
- Product category performance
