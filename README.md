# 📌 Project Overview

This project focuses on analyzing customer purchasing behavior and sales trends using the **Instacart Dataset**.

The analysis was performed using:
- Python
- SQL
- Power BI
- Google Colab

The main objective of this project is to uncover:
- Customer shopping patterns
- Product demand trends
- Reordering behavior
- Peak purchasing hours
- Sales insights for business decision-making

---

# 🎯 Project Objectives

- Analyze customer order behavior
- Identify most ordered products
- Understand reorder frequency
- Detect peak order timings
- Study customer purchasing trends
- Build an interactive Power BI dashboard
- Generate business insights using analytics

# 🔍 Key Insights

## 🥇 Most Ordered Products
- Banana was identified as the most ordered product
- Organic food items dominate top purchases

## ⏰ Peak Ordering Hours
- Orders significantly increase during daytime hours
- Peak activity observed between morning and afternoon

## 🔄 Reordering Behavior
- Frequently reordered products indicate strong customer loyalty
- Organic and daily-use products show high reorder rates

## 📅 Orders by Day
- Weekend and early-week orders show higher activity
- Customer ordering behavior varies across weekdays
